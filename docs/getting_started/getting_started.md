# Getting started guide

## Installation

```console
pip install viadot2
```

!!! info

    You can also install `viadot2` with any of the [supported extras](https://github.com/dyvenia/viadot/blob/2.0/pyproject.toml) with `pip install viadot2[extra]`, eg. `pip install viadot2[azure]`
